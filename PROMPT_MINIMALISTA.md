# 💡 Prompt Minimalista para futuros cambios

Usa esto cuando necesites cambios rápidos sin explicaciones largas:

---

## Cambios pequeños:
```
Proyecto: Camera Monitor (Next.js, Vercel)
Cambio: [lo que quieres cambiar]
Archivo: pages/index.js (o el que sea)
Línea: [número si sabes]
Especificar: [detalles técnicos si hay]
```

**Ejemplo:**
```
Proyecto: Camera Monitor
Cambio: Cambiar color de alerta de rojo a naranja
Archivo: styles/Dashboard.module.css
Buscar: #FCEBEB y #F09595
Reemplazar con: #FFF3E0 y #FFB74D
```

---

## Deploy/Config:
```
Acción: [update/deploy/config]
Detalle: [qué hacer]
```

**Ejemplo:**
```
Acción: update
Detalle: Cambiar URL de API a https://nueva-api.com
```
